# Добавление зеркальной рамки к изображению

Приложение на Tkinter и OpenCV для автоматического расширения границ изображения путем зеркального отображения его краев. Программа позволяет динамически регулировать толщину рамки и сохранять полученный результат.

## Возможности

- **Загрузка изображений** популярных форматов (`.jpg`, `.png`, `.bmp`).
- **Интерактивное изменение размера полей** от 10 до 60 пикселей с помощью ползунка.
- **Зеркальное отражение (BORDER_REFLECT)**, что позволяет незаметно увеличивать холст для печати или дальнейшей обработки.
- **Информационный блок**, отображающий исходный и итоговый размер изображения в пикселях.
- **Экспорт** готового результата в форматы PNG или JPEG.

## Установка зависимостей

python3 -m venv .venv
source .venv/bin/activate 
pip install opencv-python pillow

## Запуск

python 24.py
